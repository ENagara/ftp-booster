# ftp-booster

## Overview
FTP向上のための記録ツール

## Description
ReactNative / Expo / Firebase を利用したクロスプラットフォーム開発の勉強をするために作成したアプリです。<br><br>

＜利用ツール＞<br>
Firebase：機能で認証・データベース・ホスティング<br>
Sentry：異常系が発生した場合などのログを収集<br>
Google Analytics：アクセスやデータ分析<br>
